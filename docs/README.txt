All files that can be modified for the case study:
    tagset              `org\languagetool\resource\tl\tagset.txt`
    grammar             `org\languagetool\rules\tl\grammar.xml`
    Tagalog             `org\languagetool\language\Tagalog.class`
    Vocabulary          `docs\raw.txt`
    Filipino_Language   `Filipino_Language\`

Commands:
    LT_ENCODE   Encoding of vocaulary
    LT_DECODE   Decode dictionary file to make into human-readable text file
    LT_RUN      Run the jar file of the LanguageTool
    TAG_CHECK   Use LanguageTool with the Java grammar rules defined in `Filipino_Language`.

Adding a rules:
    Read README.md section `Creating your own Java Grammar Rules`
